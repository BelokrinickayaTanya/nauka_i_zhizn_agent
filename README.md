# 🔬 Агент журнала «Наука и жизнь»

## 📌 Описание проекта

**RSS-агент с логическими рассуждениями** для поиска и анализа научно-популярных статей журнала «Наука и жизнь».

Что делает агент:
- 🔍 **Ищет статьи** по запросу пользователя (BM25 + лемматизация)
- 🧠 **Показывает рассуждения** (Chain-of-Thought)
- 📊 **Сообщает количество** найденных статей
- 📰 **Отображает последние новости** из RSS-ленты

---

## 🎯 Цель проекта

Разработать агентную систему для поиска научно-популярных статей журнала «Наука и жизнь» с использованием RSS-ленты как внешнего источника данных.

**Задачи:**
1. Реализовать агента с одной тулзой (RSSReader)
2. Внедрить поиск с ранжированием (BM25 + лемматизация)
3. Добавить Chain-of-Thought для прозрачности рассуждений
4. Создать веб-интерфейс на Streamlit

---

## 🔧 Используемые технологии

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Поиск** | BM25 (rank_bm25) | Ранжирование статей по релевантности |
| **Морфология** | pymorphy3 | Приведение слов к нормальной форме |
| **RSS** | feedparser | Парсинг RSS-лент |
| **Интерфейс** | Streamlit | Веб-интерфейс |
| **Логирование** | JSON + datetime | Сохранение действий и метрик |

---

## 🛠️ Обоснование выбора технологий

| Технология | Почему выбран |
|------------|---------------|
| **BM25** | Промышленный стандарт поиска, не требует обучения |
| **pymorphy3** | Лучшая лемматизация для русского языка |
| **feedparser** | Стандарт для парсинга RSS на Python |
| **Streamlit** | Минимальный код для веб-интерфейса |

## Использование
Поле ввода — напишите запрос (например, «археология», «африканские рыбы»)

Кнопка «Искать» — запуск поиска

Кнопка «Последние новости» — просмотр всех статей из RSS

Chain-of-Thought — раскрывающийся блок с рассуждениями агента
